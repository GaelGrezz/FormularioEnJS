## Objetivo general
Desarrollar un formulario maquetado en HTML, estilizado en CSS y validado en JS para simular el envío de correos. 

## Propósito
Dominar el uso de JavaScript para el desarrollo de formularios seguros y eficientes; implementando código limpio y traversing para crear elementos en el DOM. 

## Resultados
Se ha implementado medidas de validación accediendo a los valores de los input mediante eventos "blur" para reconocer si un elemento carece de valor. 

Se ha implementado una clase para controlar la impresión de mensajes en el DOM y centralizar el código.

Se implementan funciones declarativas para validar, controlar y verificar el contenido de los inputs.
